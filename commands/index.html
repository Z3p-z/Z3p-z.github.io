Melo Commands
Public • Safe • Filtered
General
help
Shows an overview of Melo’s main features and command categories.
ping
Displays the current bot latency and basic status.
info
Shows information about Melo, version and links.
Moderation
warn
Warns a user and logs the warning (if enabled).
mute
Temporarily mutes a user according to your server rules.
clear
Clears a specified number of messages in a channel.
Utility
avatar
Shows the avatar of a user in high resolution.
serverinfo
Displays information about the current server.
userinfo
Shows information about a specific user.
Fun
8ball
Ask a question and get a random answer.
meme
Sends a random meme from a curated source.
