Bereitstellung „as is“

Melo wird „as is“ bereitgestellt. Das bedeutet, dass der Bot ohne Garantie, ohne Anspruch auf Vollständigkeit und ohne Zusicherung bestimmter Funktionen genutzt werden kann. Änderungen, Updates oder Abschaltungen können jederzeit erfolgen.

Welche Daten verarbeitet Melo?

Melo verarbeitet ausschließlich Daten, die für die Funktion des Bots notwendig sind. Dazu gehören:

• Discord-Nutzernamen
• Discord-IDs
• Nachrichteninhalte, sofern sie für Commands benötigt werden
• Server-Informationen, die Discord automatisch übermittelt

Es werden keine Daten an Dritte weitergegeben.

Speicherung von Daten

Melo speichert nur Daten, die für Economy-, Moderations- oder Statistikfunktionen notwendig sind. Diese Daten können jederzeit auf Anfrage gelöscht werden.

Kontakt

Bei Fragen zur Datenverarbeitung oder zur Löschung gespeicherter Daten kannst du das Melo-Team über den Discord-Supportserver kontaktieren.
